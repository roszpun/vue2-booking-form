# A Recruitmen assignmnt for eobuwie.pl

## Getting started
```
git clone https://github.com/roszpun/vue2-booking-form.git
```
```
cd vue2-booking-form
```
```
npm install
```

### Running development environment
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run unit tests
```
npm run test:unit
```
